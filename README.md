# Predictive Modelling of Signal Strength and Handover Events Using Machine Learning
A **hybrid wireless mobility prediction framework** integrating MATLAB simulation, Python XGBoost classification, and real-world 5G data.

This project demonstrates a complete end-to-end **5G handover prediction workflow,** suitable for academic research, mobility analysis, and telecom prototyping.

---

##  Project Overview

Mobile users experience handovers as they move between base stations.
Traditional threshold-based algorithms struggle under fast fading, high mobility, and overlapping cell coverage.

This project builds a **data-driven predictive handover system** using:

MATLAB for synthetic wireless simulation

Python XGBoost for handover classification

Real-world Jio True 5G measurements (GNetTrack Lite)

MATLAB GUI for interactive visualization

The system:

**1.** Generates synthetic signal strength data

**2.** Labels handover events

**3.** Trains ML models on combined datasets

**4.** Predicts and visualizes handover behavior
